# keyfire.github.io

Домен документации: **[docs.keyfire.ru](https://docs.keyfire.ru)**.

Репозиторий держит две вещи: файл `CNAME` (свой домен для GitHub Pages всей учётной
записи) и главную страницу со списком проектов. Документация каждого проекта живёт
в его собственном репозитории и публикуется оттуда:

| Раздел | Репозиторий |
|--------|-------------|
| [docs.keyfire.ru/edt-bridge](https://docs.keyfire.ru/edt-bridge/) | [keyfire/edt-bridge](https://github.com/keyfire/edt-bridge) |
| [docs.keyfire.ru/xbsl](https://docs.keyfire.ru/xbsl/) | [keyfire/xbsl](https://github.com/keyfire/xbsl) |

Поэтому здесь нет сборки: страница статическая, Pages отдаёт её прямо из ветки `main`.
