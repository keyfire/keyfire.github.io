Документация KeyFire

Открытые инструменты для разработки на платформах 1С.

EDT-Bridge 1С:Предприятие

Плагин 1C:EDT, который отдаёт живую семантическую модель среды ИИ-агентам по MCP: собственные диагностики среды, метаданные и типы, проверка запросов по метаданным проекта, правки штатным движком EDT и отладка запущенной базы.

XBSL 1С:Элемент

Инструментарий для XBSL: линтер с автоматическими исправлениями, LSP-сервер, поиск по документации платформы и заготовки метаданных, плюс расширение VS Code на том же движке.

elemctl 1С:Элемент

CLI, MCP-сервер и библиотека для Console API платформы: приложения, сборка из исходников и деплой одной командой – с честной проверкой, что изменение действительно применилось. Плюс отладчик XBSL для VS Code.
Исходники и релизы – на github.com/keyfire. О проектах и практике – keyfire.ru.
